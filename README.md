# Zero Notes

自分用に作った、メモ・日記・タスク・創作ログをまとめて保管できるメモサイトです。

## 概要

Zero Notes は、既存のメモアプリでは少し使いにくかった部分を、自分用に使いやすくするために作った個人用メモ保管庫です。

クイックメモ、タスク、日記、チャット風メモ、曲メモ、人物帳など、用途ごとに分けて記録できます。

## 主な機能

- クイックメモ
  - 1行目をタイトルとして扱う、文字だけのシンプルなメモ

- 通常メモ
  - Markdown風に書けるメモ
  - 階層フォルダで整理可能

- タスク
  - 未完了 / 完了で管理できるシンプルなタスク機能

- チャット風メモ
  - 話者を作って、会話ログのようにメモを残せる機能
  - 主役を右側、それ以外を左側に表示

- シンプル日記
  - カレンダーから日付を選んで書く日記

- 夢日記
  - 紫系のデザインで、見た夢を日付ごとに記録

- プレイリスト
  - 曲名、アーティスト、ジャンル、歌詞、和訳、URLを保存

- ストーリー置き場
  - 作品ごとに1話、2話、番外編などをまとめて保管

- 人物帳
  - リアル・キャラ問わず人物情報を記録
  - MBTI、エニアグラム、タグ、自由プロパティにも対応

## 使用技術

- HTML

- GitHub Pages

## 公開範囲について

このサイトは基本的に自分用のメモサイトです。

一部機能のみ、身内や親戚向けに使えるようにする想定です。

公開予定の機能：

- クイックメモ
- タスク

管理者専用の機能：

- 通常メモ
- チャット風メモ
- 日記
- 夢日記
- プレイリスト
- ストーリー置き場
- 人物帳
- 画像メモ

## データについて

ログインしたユーザーごとに、自分のデータだけ保存・表示されるようにしています。

Firestore ルールにより、他のユーザーのデータは読み書きできない設計です。
