# notia

**notia** は、メモ・タスク・共有メモ・創作ログなどをまとめて保管できるメモサイトです。

思いつきをすぐ残したり、やることを整理したり、自分用の記録をまとめたりするために作っています。

## サイト

https://notia.jp/

## 主な機能

### 利用できる機能

- **Home**
  - 最近書いたメモや記録をまとめて確認できます。

- **クイックメモ**
  - 思いついたことをすぐに保存できます。
  - タイトルなしでも気軽に書けるシンプルなメモ機能です。

- **タスク**
  - やることを登録して、完了したらチェックできます。
  - 軽いToDo管理に使えます。

- **共有メモ**
  - クイックメモの内容を共有用データとしてコピーし、URLで共有できます。
  - 元のメモ本体はそのまま非公開の場所に残ります。

- **アカウント設定**
  - ログイン中のアカウント情報や、使用状況を確認できます。

### 調整中・実装予定の機能

- 通常メモ
- 日記
- 夢日記
- 曲メモ
- ストーリー置き場
- 人物帳
- 画像メモ

これらの機能は、現在は調整中または管理者向けに開発中です。

## 特徴

### 自分用を大切にした設計

notia は、まず「自分が使いやすいこと」を大切にして作っているメモサイトです。

一般的なメモアプリでは使いにくい部分や、自分の記録方法に合わない部分を、自分に合う形で作っています。

### 共有はコピー方式

共有メモでは、元のメモをそのまま公開するのではなく、共有用データとしてコピーします。

そのため、元のメモ本体は自分だけの場所に残ります。

## 注意事項

重要なデータ、失うと困るデータ、長期的に残したいデータは、利用者自身でも別の場所にバックアップしてください。

また、住所・電話番号・パスワード・本人確認書類・金融情報・他人の個人情報など、外部に漏れると困る情報は入力しないでください。

詳しくはサイト内の利用規約とプライバシーポリシーを確認してください。
